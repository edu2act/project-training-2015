# 2015级项目实训成果展示 

## 《Bigdata_Movie》 - Java与大数据分析

### 项目简介

电影网站是一个专业推荐电影的网站，它囊括了大多数电影，给用户一个较为全面的推荐，也给用户提供了一个优秀的分析电影的平台。
我们团队致力于打造一个更为全面的电影推荐评价网站，让用户可以观看到更多高质量和经典的影片。在网上发表自己对电影和演员的真是意见。让用户之间可以更好地交流有关观影的心得。
